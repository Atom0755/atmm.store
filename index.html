❓ 帮助 / Help
如何添加货架和库位？▶
点击区域标签页 → 点击"+"按钮添加通道/货架，每个格位会自动生成库位编号（如 A-1-01）。点击库位编号可查看对应二维码和条形码。
库位编号格式是什么？▶
格式：J/T-区域序号-货架/排-列序号-格位序号
例：J-1-A-2-03 = 第1货架区·A货架·第2列·03格位；T-2-A-1-3 = 第2托盘区·A排·第1托盘·第3层。J=货架区，T=托盘区。
如何邀请团队成员？▶
老板账号登录 → 设置 → 团队成员 → 填写邮箱并选择角色（经理/操作员）→ 发送邀请。或将仓库邀请码分享给对方，让其注册后填入加入。
如何导出数据到Excel？▶
点击右上角"📋"图标 → 选择"导出 Excel"，即可下载当前区域的库存数据电子表格。
数据安全吗？会丢失吗？▶
所有数据存储在云端（Supabase 数据库），每次保存都会实时同步，多设备可同时访问，不依赖本地存储，不会因为清缓存丢失。
忘记密码怎么办？▶
点击登录弹窗中的"忘记密码"，输入注册邮箱，系统会发送重置链接到您的邮箱，点击链接后设置新密码即可。
如何取消或管理订阅？▶
设置 → 订阅状态 → 点击"管理订阅"，进入 Stripe 账户门户，可以查看账单历史、更改方案或取消订阅。
