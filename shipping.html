📋 获取 API 凭证步骤

  • ShipEngine（推荐聚合）: 注册 shipengine.com → Dashboard → API Keys → 免费500单/月
  • EasyPost（另一选择）: 注册 easypost.com → API Keys → 免费档可用
  • Amazon SP-API: sellercentral.amazon.com → Apps → Developer Profile → 申请（需数天审批）
  • eBay API: developer.ebay.com → Get an App ID → OAuth 应用（当天批准）
  • USPS 地址验证: registration.shippingapis.com → 申请免费 API key
